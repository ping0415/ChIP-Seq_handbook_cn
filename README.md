## 初衷

我阅读学习这本书的时候是2020年3月，也是因为自己有ChIP-Seq的分析任务，所以想找一些比较好的资料学习学习。看原版教程是我比较喜欢的方式，感觉就时刻在于作者进行沟通，顺着他们的思路一步步往下走。在学习的过程中，我也加入了自己的理解以及其他相关的知识，因此这一本也不会是简单的翻译版本，包括其中的代码我也进行了优化，更加容易操作。
